# ESPhomeMonitor Boards


# Boards
<img src="https://github.com/palmerr23/ESPhomeMonitor/blob/main/Gallery/main_board.png" width=500>
+ Main: Pico W, 3 x peak reading analogue inputs (400mV - 48V), filtered PWM output.
![ESPHome Main Board](/Gallery/main_board.png "ESPHome Main Board")
+ Analog Input: 8 x analog inputs (3.5V - 48V) 3.81mm pluggable terminals or IDC connector.
![Analog IDC](/Gallery/analog_IDC.png "Analog IDC")
![Analog Pluggable](/Gallery/analog_pluggable.png "Analog Pluggable")
+ Analog Input (RVR): A specialised analog input board for RVR TEX150 transmitters and TXRL/RXRL STL.
![Analog RVR](/Gallery/analog_RVR.png "Analog RVR")
+ Digital Input: eight channels, opto-isolated or non-isolated. 3.81mm pluggable terminals or IDC connector.
![Digital input IDC](/Gallery/digital_in_IDC.png "Digital input IDC")
![Digital input pluggable](/Gallery/digital_in_pluggable.png "Digital input pluggable")
![Digital input isolated IDC](/Gallery/digital_in_iso_IDC.png "Digital input isolated IDC")
![Digital input isolated pluggable](/Gallery/digital_in_iso_pluggable.png "Digital input isolated pluggable")
+ Digital Output: eight channels, 4 x opto-isolated + 4 x relays, or non-isolated. 3.81mm pluggable terminals or IDC connector.
![Digital output](/Gallery/digital_out.png "Digital output")
+ Expander Board: Second stack and additional 3.3V regulator.
![Extender](/Gallery/extender.png "Extender")
